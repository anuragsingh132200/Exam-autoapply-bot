# Exam Automation Frontend

Next.js admin dashboard for the exam automation platform.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Dashboard with stats and quick actions
- `/exams` - Manage exam configurations
- `/users` - Manage user profiles
- `/workflow` - Run automation with live view

## API

Proxies to backend at `http://localhost:8000`
